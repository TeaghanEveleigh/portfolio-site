NOTES.AI

Notes.ai is a webapp that leverages the power of ai apis in order to generate notes based of users description alternatively the user can just store their own to do notes that are not ai generated. Made with ejs and hosted on haroku

  • stores user information by login details using MongoDB
  • Generates to notes based of user description
  • allows user to save and store their to notes for later access
  • uses ejs to create a UI
  • AI api sends request to api and responds with a text description and this is then decomposed into a note

Book keeper Webapp

Web app that allows users to keep track of all their books!

  • users can track what they are currently reading
  • Makes use of googles book API
  • Users can add books to their libraries that they create
  • Users can search for books and add them to their library
  • backend done with express using bcrypt for secure logins
  • SQL database used for storing users information

BNZ Banking

BNZ banking is a clone of the BNZ banking app/ made with C++ the project uses various elements of the c++ STL library and makes use of modern c++ 11 features. This project is a command line interface the users information is stored using a hashmap algorythm and the user has the ability to:

  • View their account balance
  • View their transaction history
  • Make a deposit
  • Make a withdrawl
  • Make a transfer
  • Add friends to a quick pay list
  • View their account information
