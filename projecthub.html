BNZ Banking

BNZ banking is a clone of the BNZ banking app/ made with C++ the project uses various elements of the c++ STL library and makes use of modern c++ 11 features. This project is a command line interface the users information is stored using a hashmap algorythm and the user has the ability to:

  • View their account balance
  • View their transaction history
  • Make a deposit
  • Make a withdrawl
  • Make a transfer
  • Add friends to a quick pay list
  • View their account information

Ai Todo list

Ai Todo list is a webapp that leverages the power of ai apis in order to generate to do list based of users description alternatively the user can just store their own to do list that are not ai generated. Made with react using x database and x server

  • stores user information by login details in a database
  • Generates to do list based of user description
  • allows user to save and store their to do list for later access
  • uses react to create a UI
  • AI api sends request to api and responds with a text description and this is then decomposed into a list which the user has acess too
